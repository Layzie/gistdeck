# GistDeck

GistDeck is a presentation layer for one of our favorite text sharing tools.

## Instructions

* <a href="https://gistdeck.herokuapp.com">View homepage</a>
* Drag to Bookmarks Bar GistDeck
* Visit Gist
* Click Bookmarklet
* Present
